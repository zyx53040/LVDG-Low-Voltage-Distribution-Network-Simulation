# 低压配电网 MATLAB 仿真模型

## 1. 项目简介

本项目基于 MATLAB R2024a 构建低压配电网仿真模型，面向台区级配电系统运行分析与数据采集验证场景。模型包含 **2 个台区**、**多个表箱**以及 **16 个用户节点**，能够实现对用户日功率负载曲线的脚本化赋值，并按照设定采样周期自动采集各类运行数据。

仿真模型重点覆盖低压配电网中用户侧负荷时变特性、表箱分层接入关系、台区变压器运行状态以及分时数据采集逻辑。系统可实现对每个用户电表 **每 15 分钟采集一次有功功率 P、无功功率 Q 和电压 V 数据**，同时支持同步采集台区变压器侧三相电压等关键运行参数，为配电网运行监测、负荷分析、状态评估和算法验证提供基础仿真平台。

---

## 2. 模型功能

本项目主要实现如下功能：

1. 构建低压配电网拓扑模型，包含 2 个台区、多个表箱及 16 个用户。
2. 通过 MATLAB 脚本对用户日负荷曲线进行统一管理与批量赋值。
3. 支持按照时间序列驱动负荷变化，实现全天运行过程仿真。
4. 实现用户电表侧按 15 分钟间隔采集 P、Q、V 数据。
5. 实现台区变压器侧三相电压同步采集。
6. 支持输出仿真结果，用于后续可视化分析、算法研究和数据集构建。

---

## 3. 模型结构说明

本模型采用“台区—表箱—用户”分层结构进行组织。

### 3.1 台区层

系统共包含 **2 个台区**，每个台区由配电变压器供电。台区层主要用于描述变压器低压侧运行特性，并作为下游表箱和用户负荷的汇聚节点。

### 3.2 表箱层

每个台区下设置若干表箱，用于组织和管理多个用户接入关系。表箱层用于反映低压配电网中用户集中接入的实际形式，并为后续扩展支路参数、台区不平衡分析和台账映射提供接口。

### 3.3 用户层

系统共配置 **16 个用户**。每个用户均可通过脚本分配独立的日负荷曲线，从而反映不同类型用户在全天不同时段的有功和无功功率变化特征。用户侧仿真结果可映射为智能电表采集数据。

---

## 4. 数据采集内容

本项目设计了面向配电监测场景的数据采集机制，采样粒度为 **15 分钟**，即每天共形成 96 个采样点。

### 4.1 用户电表采集数据

对每个用户，仿真过程中按 15 分钟间隔采集如下参数：

- 有功功率 P
- 无功功率 Q
- 电压 V

上述数据可用于负荷曲线分析、用户用电行为研究、电压质量评估以及异常检测算法测试。

### 4.2 变压器侧采集数据

在台区变压器侧，同步采集三相电压数据，包括：

- A 相电压
- B 相电压
- C 相电压

该部分数据可用于台区供电质量分析、电压偏差研究及三相不平衡特性评估。

---

## 5. 负荷曲线赋值方式

项目采用 MATLAB 脚本对用户日负荷曲线进行集中控制。典型流程如下：

1. 预先定义 96 点日负荷曲线数据。
2. 根据用户编号或用户类型，将对应负荷曲线映射到各个用户节点。
3. 在仿真运行过程中，按照时间步长逐点更新各用户负荷值。
4. 完成全天功率时序驱动仿真。

该方式具有结构清晰、可维护性强、便于批量测试等优点，适用于后续扩展典型日、季节性负荷、随机扰动负荷以及光伏接入等场景。
